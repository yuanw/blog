I recently started to use starlight.astro.build for writing this blog.

# Why Starlight
The biggest things I feel like out of box Hakyell missing is [sidenote](https://tony-zorman.com/posts/block-sidenotes.html).

starlight offers lots of cool components for writing documentations:

https://starlight.astro.build/guides/components/
## Sidenote / Asides

:::tip
It’s best to avoid using `<blockquote>` for things that aren’t quotes.
:::
