learning nix for haskell

Posted on July 28, 2019 by Yuan Wang

Environment

  • nix version: 2.2.2
  • os: macOS 10.14.5

Install nix

I removed ghc (installed using ghcup), cabal beforehand.

curl https://nixos.org/nix/install | sh this would install single-user mode nix on the machine. All the executables would installed $HOME/.nix-profile/bin/. The installsion would created the following directories: $HOME/.nix-channels, $HOME/.nix-defexpr, /nix.

It seems nix supports bash or zsh. For using nix in fish, maybe checkout this.

We can verify the install by type nix --version

update nix

Basically, for single-user installations, we can run nix-channel --update; nix-env -iA nixpkgs.nix
